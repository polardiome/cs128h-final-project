Project Overview:

Likeminded: A Guessing Game

Modelled from Wavelength, Likedminded is a two-player guessing game where players can see how likeminded they are about various topics of their choice. For example, if the topic is length, the first player would be randomly given an index on a scale from 0 to 11, and they would think of something that would match the given index. In this case, 0 could represent short and 11 long. If the player is given 10, they might think of a giraffe because it is long. The second player would guess the correct index based on giraffe alone. If they guess the index exactly, they earn three points. If the guess one away from the index, they earn two points, and so on...

There are also special bonuses in the game. If the player guesses exactly multiple times in a row, they can earn extra points. Similarly, some turns may be double points on a random basis.

After every guess, the players switch roles.

The game ends after a certain number of rounds based on what the players choose at the beginning. The goal of the game is to get the highest score possible collectively by knowing each others' thought processes well.
