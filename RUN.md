Likeminded Instructions

Getting Started:
1. git clone https://github.com/polardiome/cs128h-final-project
2. cargo run

Playing the Game:
1. Player 1 starts by choosing descriptions for a set of lower and upper bounds
2. Based on a random number, Player 1 chooses a guess word for Player 2
3. Player 2 guesses the correct number based on the guess word and bounds descriptions
4. A correct guess is worth 3 points, one number away is worth 2 points, and two numbers away is worth 1 point
5. Players switch turns and play for ten rounds
